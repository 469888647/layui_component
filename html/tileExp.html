快捷菜单

快捷菜单组件tile是对WIN10-UI功能的扩展:在开发文档的未来计划中,作者想要实现可拖拽磁贴的功能。

文件依赖

以下的功能是对模块tile的描述,它除了需要tile.js文件之外,还需要引用下面的内容。

css文件引用

  • 在引用了layui.css后,还需要引用tile.css。当然,如果是当前分支进行打包tile.css就打包进了layui.css中,不用额外引用了

  • 在引用了layui.css后,还需要引用第三方的animate.css。当然,如果是当前分支进行打包animate.css就打包进了layui.css中,不用额外引用了

js文件引用

在引用了layui.js后,还需要注意下面模块的加入。当然,如果是当前分支进行打包就包括下面的模块了(除开第三方插件)。

  • 模块jquery
  • 模块layer
  • 模块dropdown可能会使用

基础介绍

为了方便维护, tile功能从原仓库COPY到这里来。

并进行了下面几点的修改:

  1. 修改成多个 快捷菜单实例的并存的模式
  2. 实现了磁贴比例和快捷菜单列数的自定义,打破之前定死的100像素单位和固定3列
  3. 将磁贴 点击事件 通过layui事件托管,不必出现在配置项中,主要针对的是配置项的缓存问题,function无法缓存
  4. 修改颜色样式,应当随着layui主题变化,主要是结合新增模块colortheme来实现
  5. 磁贴配置项中新增content,方便调用者进行自定义展示信息(如嵌入流媒体内容)
  6. 磁贴配置项添加kv属性,用于补充信息;id可以不用传了,防止传入重复的id导致系统出问题
  7. 新增 磁贴右键事件,弹出dropdown菜单来实现删除功能
  8. 新增了一个 缓存更新事件 ,用于缓存更新时返回缓存信息来进行后端的持久化操作
  9. 新增tile参数 refuseAnimate ,默认false 是否禁用动画(true-禁用;false不禁用)
  10. 修复了一个bug,这会导致小磁贴在左右移动时会叠在大磁贴上面

属性与方法

tile中提供的方法现在被设计成类似jquery库的风格,可以满足链式调用。

菜单渲染

layui.tile(destination, option);或者new layui.tile(destination, option);

  • 参数destination必填,指定一个渲染的dom区域,类型String/DOM

    请注意:指定的这个dom最好

    1. 加上classlayui-layer-tiles设置成position: absolute
    2. 加上classlayui-windows-desk-scroll设置滚动条样式
    3. 再自行指定一个具体的高度和宽度

  • 参数option可选,是一个配置项,通过设置不同的参数来生成不同的菜单

    key 类型 描述 默认值
    id String (选填) 当前菜单的唯一编号,除非你要固定的使用缓存信息,否则建议不填随机抽取一个即可 {{constant.INSTANCEID}}
    columns Number (选填) 当前菜单的最大列数,将会同步给下面的块对象 3
    capacity Number (选填) 当前菜单的单位像素点,将会同步给下面的块对象 100
    cacheable Boolean (选填) 当前实例是否允许开启缓存(true-开启缓存;false禁用缓存) false
    data Array (必填) 当前菜单的内容配置项,内容为块的配置项 /

  • 返回值 tileInstance被创建好的菜单实例对象

data 格式 : data是一个数组,数组的元素是块的配置项;块的配置项里面有一个source,它也是一个数组,数组的元素是磁贴的配置项。

块的配置项

key 类型 描述 默认值
id String (选填) 当前块的唯一编号 {{constant.STRUCTID}}
name String (选填) 名称,展示在title上面,后期可以允许用户自定义 ''
source Array (必填) 当前块的内容配置项,内容为磁贴的配置项 /

磁贴的配置项

key 类型 描述 默认值
id String (选填) 当前磁贴的唯一编号 {{constant.TILEID}}
name String (选填) 名称 ,与 img、 content 配置项需要至少传入一个 ''
img String (选填) 引用图片路径 ,与 name、 content 配置项需要至少传入一个 ''
content String (选填) html片段 ,与 name、 img 配置项需要至少传入一个 ''
bgColor rgba、hex等颜色字符串 (选填) 背景颜色 ,默认取layui主题颜色 /
color rgba、hex等颜色字符串 (选填) 文字颜色 ,默认取layui主题颜色 /
x Number (选填) 磁贴的横坐标 0
y Number (选填) 磁贴的纵坐标 0
w Number (选填) 磁贴的宽度系数 1
h Number (选填) 磁贴的高度系数 1
refuseAnimate Boolean (选填) 是否禁用动画(true-禁用;false不禁用) false
kv Object (选填) 补充参数,用于自定义保存内容 {}

事件

on(type, callback);可以为实例添加监听事件,请注意该方法通过tileInstance实例调用

  • 参数 type 是绑定事件的类型, 当前版本支持 clickTile 磁贴点击事件 和 updateCache 缓存更新事件

  • 参数 callback 是一个回调函数,在上面事件触发时,会调用这个函数进行业务处理

  • 返回值 tileInstance 菜单实例对象

点击事件

on('clickTile', callback);可以为实例添加磁贴点击事件,请注意该方法通过tileInstance实例调用

  • 参数 callback 是一个回调函数,在上面事件触发时,会调用这个函数进行业务处理

    这个函数接收一个 option 的变量,这个变量包含下面的内容:

    1. struct: 当前块的信息
    2. tile: 当前磁贴的信息

  • 返回值 tileInstance 菜单实例对象

缓存更新事件

实验 on('updateCache', callback);可以为实例添加缓存更新事件,请注意该方法通过tileInstance实例调用

  • 参数 callback 是一个回调函数,在上面事件触发时,会调用这个函数进行业务处理

    这个函数接收一个 option 的变量,这个变量包含下面的内容:

    1. key: 当前使用的缓存key,用于不同实例之间的区分
    2. data: 当前缓存的信息,包含所有的配置项信息

  • 返回值 tileInstance 菜单实例对象

缓存更新事件作为一个保留功能,这里暂时不提供示例

新增磁贴项

提供两种方式新增磁贴项,但是更加建议初始化的时候就将配置项处理好

添加磁贴项

实验 addTile(tileOption);可以新增一个块在上面创建一个磁贴,请注意该方法通过tileInstance实例调用

  • 参数 tileOption 是磁贴的配置项

追加磁贴项

实验 insertTile(structId, tileOption);可以在指定的块上面追加一个磁贴,请注意该方法通过tileInstance实例调用

  • 参数 structId 是块配置项的 id 属性,如果不清楚也可通过上面的方法添加，然后拖到指定的块中

  • 参数 tileOption 是磁贴的配置项

点击下面的按钮可以发现:

  1. 添加磁贴项每次都会创建一个新的块
  2. 追加磁贴项每次都是在指定的layui这个块下面新增磁贴
  3. 由于磁贴配置项没有指定id,通过点击可以看出每一个磁贴的id都是唯一的

设置动画

setAnimate(animated_classes, animated_liveness);可以为实例添设置动画参数,请注意该方法通过tileInstance实例调用

  • 参数 animated_classes 是动画的类型, 传入类型是字符串的数组 {Array[String]}, 支持下面几个类型:

    • flip
    • bounceIn
    • flash
    • pulse
    • rubberBand
    • shake
    • headShake
    • swing

  • 参数 animated_liveness 是一个动画系数,取值范围是 [0,1] ,值越大动画越频繁

  • 返回值 tileInstance 菜单实例对象

缓存操作

缓存操作先是要开启/关闭缓存,然后再设置是否自动缓存。自动缓存是指在配置项信息更新时将最新的信息更新到缓存中。

使用缓存

实验 enableCache(), 可以开启缓存的功能。它与构造函数配置项里面的 cacheable 属性的区别是：后者会在初始化时使用缓存信息对实例进行初始化。

禁用缓存

实验 disableCache(), 可以禁用缓存的功能。

自动缓存

实验 enableAutoCache(), 可以在配置项信息更新时将最新的信息更新到缓存中。

清空当前缓存

实验 resetData(), 可以将当前实例的缓存信息清除。

面板操作

块拖动

鼠标放在快捷菜单中的一个块上面,右上方会出现一个小图标。按住标题可以将整改块的位置向上或向下拖动

磁贴拖动

鼠标放在快捷菜单中的一个磁贴上面,按住可以将它的位置进行拖动。如果将它拖动在最下方没有块的位置,那个位置会新建一个块。

编辑块标题

鼠标点击块标题的位置就会出现输入框,输入后失去焦点就编辑成功了

编辑磁贴

鼠标放在快捷菜单中的一个磁贴上面,右键可以呼出dropdown菜单。这里可以修改磁贴的尺寸和删除这个磁贴。

自定义内容

当前磁贴配置项中新增content配置项,可以通过插入html的方式引入其它的插件。
