项目打包

Layui 提供了 gulpfile.js 和相关依赖包，在 gitee 拉下整个仓库代码，再安装 Gulp 进行构建。

这里沿用此方式:

  • 1、在本地文件下面右键菜单执行Git Bash Here,需要先安装git.
  • 2、在git窗口中输入git clone -b layuiframework https://gitee.com/giteetcj/layui.git,拉取指定的分支.
  • 3、进入项目文件夹(名为layui),在这个文件夹下面打开cmd窗口.
  • 4、输入gulp命令进行打包(如果提示输入npm install需要先执行这个命令),打包好的文件在dist文件夹下面.
  • 5、输入gulp release创建发行版本,打包好的文件在release文件夹下面.
